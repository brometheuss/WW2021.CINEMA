# Introduction 
Cinema 9 is client application that consumes WEB API created on LEVI9 WINTER WORKSHOP 2020. Application is created using ReactJs java script library.

# Getting Started - VS Code
To start using application on your machine, you need to follow this steps.
1.	Install NodeJs and NPM (Node Package Manager - comes with Node) from [NodeJs website](https://nodejs.org/en/)
2.	Clone or download this repository
3.	Open Cinema9 application solution and navigate to cinema folder 
4.	Open folder in Visual Studio Code
5.  In VS Code start new terminal and type *npm install*
6.  After installation is complete type *npm start*
7.  In your browser navigate to [http://localhost:3000](http://localhost:3000)
